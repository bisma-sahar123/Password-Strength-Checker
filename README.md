# 🔒 Password Strength Checker

## Description

The Password Strength Checker is a Python application developed as part of the DecodeLabs Cyber Security Internship. This project analyzes the strength of a password by checking important security factors such as length, uppercase and lowercase letters, numbers, and special characters.

Based on these checks, the application classifies the password as Weak, Medium, or Strong and provides helpful feedback to improve password security.

## Features

- Check password strength
- Detect weak, medium, and strong passwords
- Suggest ways to improve password security
- Easy-to-use command-line interface

## How to Run

1. Make sure Python 3 is installed on your computer.
2. Download or clone this project.
3. Open the project folder in Visual Studio Code.
4. Open the terminal.
5. Run the following command:

```bash
python password_strength_checker.py
```

6. Enter a password when prompted, and the program will evaluate its strength.
